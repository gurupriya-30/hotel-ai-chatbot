Hotel Chatbot with Flask

A beginner-friendly Hotel Chatbot built with Python and Flask that allows users to interact with a virtual hotel assistant to check room details, facilities, prices, and make booking inquiries.
This project demonstrates a simple rule-based chatbot integrated into a web application.


Project Overview:

The Hotel Chatbot helps users:
     Get information about different room types (Single, Double, Deluxe).
     Know room availability and pricing.
     Learn about hotel facilities, check-in/out times, and location.
     Ask about food services or contact details.
     Receive guidance on booking procedures.
     The chatbot uses Python logic to process user queries and Flask to provide a simple web interface.

Features:

     Interactive chatbot interface using Flask and HTML templates.
     Handles multiple user queries about hotel services.
     Simple rule-based responses that are easy to extend.
     Beginner-friendly code suitable for learning web-based chatbots.

Technologies Used:

   Python – Programming language.
   Flask – Web framework to serve the chatbot.
   HTML/CSS – Frontend templates for user interface.
   Jinja2 – Flask templating engine for dynamic content.
